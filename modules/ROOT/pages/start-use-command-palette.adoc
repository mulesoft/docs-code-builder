The Command Palette in Anypoint Code Builder is the control center for all commands you use to perform tasks. Knowing how to open it is important for your productivity.

. Open the Command Palette in one of the following ways:

* Use the keyboard shortcuts:
** Mac: Cmd+Shift+p
** Windows: Ctrl+Shift+p
* In the desktop IDE, select *View* > *Command Palette*. 
* In the cloud IDE, click the image:icon-menu.png["",18,18] (menu) icon, and select *View* > *Command Palette*:
+
image::open-command-palette.png[]

. Type `MuleSoft:` and browse the available commands:
+
image::all-mulesoft-commands.png[]

For a list of MuleSoft commands, see xref:ref-acb-commands.adoc[].
